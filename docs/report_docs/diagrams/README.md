# AI Customer Support Agent - Diagrams

This folder contains all diagram code for the project report.

## How to Render Diagrams

### Option 1: Online Tools
- **PlantUML**: https://www.plantuml.com/plantuml/uml/
- **Mermaid Live**: https://mermaid.live/

### Option 2: VS Code Extensions
- Install "PlantUML" extension by jebbs
- Install "Mermaid Preview" extension

### Option 3: Draw.io Import
- Some diagrams can be recreated in Draw.io using the specifications

## Files in this folder:
1. `use_case_diagrams.puml` - All Use Case Diagrams (PlantUML)
2. `sequence_diagrams.puml` - All Sequence Diagrams (PlantUML)
3. `class_diagram.puml` - Class Diagram (PlantUML)
4. `architecture_diagram.md` - System Architecture (Mermaid)
5. `dfd_diagrams.md` - Data Flow Diagrams (Mermaid)
6. `er_diagram.puml` - Entity Relationship Diagram (PlantUML)

## Rendering Instructions

1. Copy the code from each file
2. Paste into the online tool or VS Code
3. Export as PNG/SVG
4. Insert into your Word/PDF document
